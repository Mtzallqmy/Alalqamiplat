# تصميم تطبيق معتز العلقمي

## نظرة عامة
تطبيق جوال لمنصة محتوى عربية بتصميم بيج/كريم مع لمسات ذهبية وواجهة RTL كاملة.

## الشاشات الرئيسية

### 1. Home (الصفحة الرئيسية)
- **المحتوى الأساسي:**
  - Featured Post: منشور مميز بصورة كبيرة وعنوان
  - Latest Posts: قائمة آخر المنشورات (3-4 منشورات)
  - Categories: أيقونات الفئات الرئيسية
  
- **الوظائف:**
  - Pull-to-refresh لتحديث المحتوى
  - التنقل إلى تفاصيل المنشور عند الضغط
  - الانتقال إلى قائمة الفئات الكاملة

### 2. Posts (قائمة المنشورات)
- **المحتوى:**
  - قائمة منشورات مع pagination
  - كل منشور يعرض: صورة صغيرة، عنوان، اسم الكاتب، التاريخ
  - Pull-to-refresh
  - Loading/Empty states
  
- **الوظائف:**
  - التمرير اللانهائي (infinite scroll) أو pagination
  - الانتقال إلى التفاصيل عند الضغط

### 3. Post Details (تفاصيل المنشور)
- **المحتوى:**
  - صورة المنشور
  - العنوان
  - معلومات الكاتب والتاريخ
  - وقت القراءة المتوقع
  - محتوى HTML آمن
  - أزرار: مشاركة، إضافة للمفضلة
  
- **الوظائف:**
  - عرض محتوى HTML بشكل آمن
  - حفظ في المفضلة (AsyncStorage)
  - مشاركة المنشور
  - التنقل بين المنشورات

### 4. Categories (الفئات)
- **المحتوى:**
  - شبكة الفئات
  - اسم الفئة وعدد المنشورات
  
- **الوظائف:**
  - الانتقال إلى منشورات الفئة

### 5. Category Posts (منشورات الفئة)
- **المحتوى:**
  - قائمة منشورات الفئة المحددة
  - نفس تنسيق قائمة المنشورات
  
- **الوظائف:**
  - Pull-to-refresh
  - Pagination

### 6. Search (البحث)
- **المحتوى:**
  - حقل بحث
  - نتائج البحث (قائمة منشورات)
  - حالات فارغة/تحميل
  
- **الوظائف:**
  - البحث في الوقت الفعلي
  - عرض النتائج

### 7. Favorites (المفضلة)
- **المحتوى:**
  - قائمة المنشورات المفضلة
  - إزالة من المفضلة
  
- **الوظائف:**
  - عرض المفضلة المحفوظة محليًا
  - حذف من المفضلة

### 8. More/About (المزيد/حول)
- **المحتوى:**
  - معلومات التطبيق
  - معلومات المنصة (من API)
  - إعدادات التطبيق
  
- **الوظائف:**
  - عرض معلومات من API

## التنقل (Navigation)
- **Tab Bar (5 تبويبات):**
  1. Home - الصفحة الرئيسية
  2. Posts - المنشورات
  3. Search - البحث
  4. Favorites - المفضلة
  5. More - المزيد

### الخطوط
- **Font Family:** Tajawal (خط عربي حديث)
- **Weights:** Regular (400), Bold (700)
- **RTL:** تفعيل كامل للنصوص العربية

## حالات التحميل والأخطاء
- **Loading:** Spinner مع نص "جاري التحميل..."
- **Error:** رسالة خطأ مع زر إعادة محاولة
- **Empty:** رسالة "لا توجد نتائج" مع أيقونة

## الميزات التقنية
- **Caching:** 5 دقائق من AsyncStorage
- **RTL:** تفعيل كامل للعربية
- **Safe Area:** معالجة آمنة للـ notch والـ home indicator
- **Pull-to-Refresh:** تحديث يدوي للمحتوى
- **HTML Rendering:** تصيير آمن للمحتوى HTML
- **Image URLs:** استخدام روابط الصور فقط (لا توجد صور محلية)

## التفاعل والـ Feedback
- **Press Feedback:** تغيير الشفافية عند الضغط
- **Haptics:** ردود حسية على الأجهزة المدعومة
- **States:** واضحة وسهلة الفهم
